== Description ==

Funny Demotivators widget add to your website  big collection of funny demotivators (demotivational posters). All pictures hosting in http://demotivators.us so you don't pay for transfer. If user click some picture he open new tab with source of picture. All pictures are checked by moderator and  will be grabbed from home page  http://demotivators.us



== Installation ==

Plugin have only one  file


Here it is in graphic form:


- wp-content
	- plugins
		- widgets
			| funny_demotivators.php
	
you install funny demotivators plugin in admin panel like other plugins. Plugin add one page to your site so if you want disable plugin you must also remove tis one page in admin panel
