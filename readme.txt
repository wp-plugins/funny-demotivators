== Description ==

Funny Demotivators let you have new page with 8 funny pictures in your site, pictures hosting in http://demotivators.us so you don't pay for transfer.
You can also choose 3 options in funny_demotivators.php default add new site with 8 random demotivators , second option add one random demotivators below all posts , third option add
mini demotivators below sidebar.This script only read random pictures from this url http://demotivators.us/widget/big_demotivators.php
or small pictures from here http://demotivators.us/widget/small_demotivators.php . All pictures are checked by modreator so 
no porn , no ofensive , no racist but you can check pictures here in home page http://demotivators.us
If user click some picture he open new tab with source of picture. Pictures look like different in different themes 
so you can simple edit css style in funny_demotivators.php




== Installation ==

Plugin have only one  file


Here it is in graphic form:


- wp-content
	- plugins
		- widgets
			| funny_demotivators.php
	
For change option in plugin open funny_demotivators.php find $option=0; , you can set 3 options 

0 - Add new page to your menu with 8 random demotivators in your blog (http://demotivators.us/widget/8_%20big_demotivators.php)
1 - Big images (width 513px) below your posts (source: http://demotivators.us/widget/big_demotivators.php)
2 - Small images (width 125px) below menu   (source: http://demotivators.us/widget/small_demotivators.php)
